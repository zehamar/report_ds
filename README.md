   pdfTeX 3.14159265-2.6-1.40.16 (TeX Live 2015/Debian)

 * KILE (ubuntu), versions: 
    Qt: 4.8.7
    KDE Development Platform: 4.14.16

 * FIG: figures
 * TEX: latex files 
 * PROD: pdf output files (different versions)
 * UTIL: miscellaneous
